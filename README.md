# AI Cognitive Health App

A full-stack web application focused on dementia awareness, cognitive wellness, and community support.

## Planned Features
- User authentication with guest access
- Daily journaling
- AI-powered journal reflections
- Cognitive games with score and time tracking
- Dashboard for trends and progress
- Community features for sharing stories and support

## Planned Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL
- Dev Tools: Git, GitHub, VS Code, Docker
- Deployment: AWS

## Project Status
Planning and environment setup in progress.
